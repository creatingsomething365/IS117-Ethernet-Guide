Inside an Ethernet Cable

Copper Strands

A single Ethernet cable holds eight copper strands, grouped into four sets. Each set wraps around itself in a spiral shape to quiet down crosstalk.

Shielding Layers

Certain versions come with added shielding layers that keep outside electrical disturbances away. This shapes how fast they work and how long they last under pressure.
